# Tiered Tridents #

## About ##

**Tiered Tridents** does exactly what you think it does - It adds more Tridents to the game in the conventional Minecraft ore tier system!

For conventional tridents, we have: Wooden, Stone, Copper, Golden, Iron, Diamond and Netherite.

And then for some curveballs, we have:

*   Pitchfork - A fire-themed Trident made out of _Blaze Rods!_
*   Bone Trident - A Trident made out of _BONES!_ Just for fun.
